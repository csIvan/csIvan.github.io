3D Spaceship Controller

Description:

A college group project that uses C++ and OpenFrameworks to create a 3D environment where the user can control a spaceship. The environment has cameras set around multiple places to view the spaceship through various angles. Using a simple raytracing algorithm, the spaceship can intersect with the terrain allowing it to land on it. Considering the terrain's large amount of vertices, we used an acceleration structure(Octree) to speed up the raytracing process and smooth out the intersections.

Achievements:

  • Designed a simple particle system and raytracing algorithm
  • Implemented the octree acceleration structure
  • Designed and modeled the 3D meshes(spaceship and terrain)

Key Components:

  • C++
  • Particle Systems
  • 3D Movement Controls
  • Octree Acceleration Structure
  • Blender
  • OpenFrameworks
