Bubble Cursor

Description:

A college group project that required us to recreate and improve upon current research related to user interface concepts. As a team of two, we chose to reproduce the bubble cursor and make an elliptical cursor that would extend based on the velocity of the cursor. We chose to use Unity and C# as our tools to make the project. I worked on establishing the bubble cursor algorithm while my partner worked on the elliptical cursor algorithm. The demo below shows our experiment.


Achievements:

  • Implemented the bubble cursor algorithm using C# and the Unity game engine
  • Conducted a formal experiment and gathered data from various individuals testing our project
  • Analyzed and formated the experiment data into graphs and diagrams using python

Key Components:

  • C#
  • Python
  • Unity
  • Visual Studio 2019
  • Firebase
