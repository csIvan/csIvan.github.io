Multiplayer Unity3D Game (Work In Progress)


Description:

A work in progress Unity3D game that uses billiard mechanics in a 1v1 environment. This is currently an independent project, but I might hire some artists later on to help me with art assest, since those are the hardest for me to properly make. The main idea of the game is to hit the opponent into the pockets/goals in a turn-based system. Both opponents can bounce off walls giving them creative opportunities to score on each other. I will update this page once I have more content to post.





Goals:

  • Develop and publish an online multiplayer versus game using C# and Unity3D
  • Integrate proper networking code for a smooth and clear multiplayer experience
  • Create and maintain a ranking system for players to compete and accomplish rewards

Key Components:

  • C#
  • Unity3D
  • Visual Studio 2019
  • Photon Networking
  • Online Multiplayer
