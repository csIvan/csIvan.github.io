Stock Viewer App

Description:

A college project for an Android mobile development course at SJSU. Worked independently to develop a mobile application that keeps track of stocks. Implemented using java and designed with Android Studio and XML. The application allows the user to search and select favorite stocks. Also, the application uses real-time information to keep the stocks up to date. The app is not published but you can still view the source code in the repository.

Achievements:
